## 比较运算符

### 用==比较数组

1. 相同维数且还有相同个数元素的数组才可以比较

2. 每个元素都相同才相等

```go
package type_test

import "testing"

func TestImplicit(t *testing.T) {
	a := [...]int{1, 2, 3, 4}
	b := [...]int{1, 3, 2, 4}
	//c := [...]int{1, 2, 3, 4, 5}
	d := [...]int{1, 2, 3, 4}
	t.Log(a == b)
	t.Log(a == d)

}

```
